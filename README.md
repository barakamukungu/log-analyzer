A Python script to analyze server log files. Counts log levels, tracks IP activity, detects failed login attempts, and summarizes logs by hour. Includes ASCII table output and optional matplotlib visualizations.

- Counts log levels: INFO, ERROR, WARNING, DEBUG
- Tracks IP address activity
- Detects failed login attempts
- Summarizes logs by hour
- Displays results as an ASCII table or matplotlib graph

#Usage

1. Clone the repository
2. Run the script: `python log_analyzer.py`
3. Enter your log filename and choose output format
